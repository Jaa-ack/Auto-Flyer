# GitHub Repo Description

English:

Auto Fly is a macOS and Windows tool for simulated iPhone GPS movement with release downloads, a browser UI, CLI, address search, and route presets.

Traditional Chinese:

Auto Fly 是一個支援 macOS 與 Windows 的工具，可透過 release 下載，並用瀏覽器介面與 CLI 進行 iPhone 模擬 GPS、地址搜尋與路線操作。
